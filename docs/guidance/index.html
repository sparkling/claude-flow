claude-flow-guidance-implementation

Production wiring for `@claude-flow/guidance` across Claude Code hooks and Codex lifecycle events, tuned for long-run reliability, policy retrieval, and measurable optimization loops.

Install From NPM Source

Quick Start

npx --yes -p claude-flow-guidance-implementation \
  cf-guidance-impl init --target ~/source/my-project --install-deps

npx --yes -p claude-flow-guidance-implementation \
  cf-guidance-impl verify --target ~/source/my-project

Modes

both claude codex

both wires Claude hooks + Codex bridge. claude and codex are platform-specific install modes.

npx cf-guidance-impl init --target ~/source/my-project --target-mode both

Core Runtime Commands

CommandPurpose
guidance:analyzeAnalyze current guidance quality and structure.
guidance:optimizeRun autopilot optimization and optionally apply changes.
guidance:autopilot:onceRun one promotion/optimization cycle.
guidance:autopilot:daemonRun continuous background optimization.
guidance:ab-benchmarkCompare outcomes with and without guidance bundles.
guidance:status, guidance:allShow integrated module and pipeline status.
guidance:trust, guidance:adversarial, guidance:proof, guidance:conformance, guidance:evolutionRun advanced governance modules directly.
