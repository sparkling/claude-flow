Advanced Module: Trust

Maintains confidence scores and policy quality signals for decisions made by runtime gates.

Advanced Module: Adversarial

Exercises risky prompts and tool calls to validate deny/warn behavior before production sessions.

Advanced Module: Proof

Builds replayable decision artifacts so restarts preserve governance decisions and auditability.

Operational Sequence

  1. Initialize with target mode and dependency install.
  2. Verify bridge wiring and hook handler syntax.
  3. Run analyze and optimize to establish baseline quality.
  4. Run codex pre/post commands (or Claude hooks) during real tasks.
  5. Track conformance and AB benchmark to validate gains.
